# TaskFlow - Project Management Dashboard

A modern, premium SaaS project management platform built with React 19, Vite, TypeScript, and Tailwind CSS.

## Features
- Kanban Board with Drag & Drop
- Project & Task Management
- Team Collaboration
- Calendar & Deadlines
- Analytics & Reports
- Dark/Light Mode
- Responsive Design

## Tech Stack
- React 19 + Vite + TypeScript
- Tailwind CSS + Framer Motion
- Zustand + TanStack Query
- Recharts + DnD Kit
